02 — Role Intent

Primary · Secondary · Accent · Decorative

Every color has a job. This keeps the palette from fighting itself: sage governs reading, rose governs action, ivory governs space, and floral accents stay in the margins.

Primary

Deep Sage

Headings, nav, primary icons, secondary buttons, link default. Carries the brand voice.

Secondary

Rose Pink

CTA background, script accents, active states, optional-tag label. Drives action + warmth.

Accent

Dusty Blush

Hover states, soft highlights, divider accents, eyebrow text. Supports without shouting.

Decorative only

Floral accents

Peach, apricot, plum, berry. Used exclusively inside watercolor artwork — never UI chrome.

D — CSS custom properties

Drop into :root and reference semantically throughout the codebase. Raw palette variables map to semantic tokens so a future theme swap is one block.

E — Accessibility

Contrast & clarity

Combination Ratio Rating Use for
#56603D on #FBF9F8 8.9 : 1 AAA Body copy on page
#667051 on #FBF9F8 6.3 : 1 AAA Headings, links, icons
#667051 on #F7F1EE 5.8 : 1 AA Headings on section band
#FFFFFF on #E67383 3.6 : 1 AA Large Primary CTA (18px+ / bold)
#FFFFFF on #667051 6.3 : 1 AAA Secondary (sage) button
#E67383 on #FBF9F8 3.2 : 1 AA Large Script accents, kickers (22px+ or decorative)
#8D8F78 on #FBF9F8 3.4 : 1 Meta only Tiny meta text, timestamps — never body

Decisions

Body copy locks to Dark Olive Sage (#56603D) on ivory. 8.9:1 contrast means paragraphs stay readable on both ivory and blush-ivory bands without tuning per section.

Rose (#E67383) is a display color, not a body color. It passes AA Large above 18px bold — which is always true for the CTA, script accents, and uppercase kickers — but it's never used for paragraph text.

Muted sage (#8D8F78) is capped at meta usage. Dates, attribution, timestamps. It fails AA for body and we enforce that by not exposing it as --text-body.

Focus rings use rose at 2.5px with 3px offset — the color doubles as brand emphasis, and the thickness carries contrast where hue alone can't.

Floral accents (peach, apricot, plum, berry) are decorative only. They live inside illustration SVGs and never carry text or interactive meaning.
